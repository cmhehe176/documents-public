🚀 Cách cài đặt Node trên Ubuntu (22.04 LTS) bằng NVM ​

🎨 Cài đặt NVM ​

📌 Để cài đặt hoặc cập nhật NVM, bạn có thể sử dụng lệnh curl để tải xuống và thực thi file. Bạn có thể tải phiên bản mới nhất từ NVM official repository

🔧 1. Cài đặt NVM ​

zsh
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.1/install.sh | zsh

🛠️ 2. Chạy lệnh cài đặt ​

zsh
source ~/.zshrc

✅ 3. Kiểm tra xem đã cài thành công hay chưa ​

zsh
command -v nvm
  • Nếu hiện lên như này là thành công
zsh
➞ command -v nvm
nvm

🌟 Cài đặt Node ​

📌 Sau khi đã cài đặt NVM, bạn có thể liệt kê tất cả các phiên bản Node.js có sẵn thông qua NVM bằng cách sử dụng ls-remote với lệnh:

🔧 1. Hiển thị danh sách version Node ​

zsh
nvm ls-remote

🛠️ 2. Cài đặt Node ​

zsh
nvm install v22.14.0

✅ 3. Kiểm tra kết quả sau khi cài đặt ​

zsh
➞ node -v
v22.14.0
